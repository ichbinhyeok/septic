# Anchor State Research Log

This file records the current homeowner-safe interpretation for the first four
anchor states. It is not meant to replace the source registry. It exists so the
editorial and product logic can see the reasoning behind the current state
profiles.

## Georgia

Why it is an anchor:

- Clear homeowner-facing bedroom logic
- Clear garbage disposal upsizing language
- Strong county health department path

Official points currently used:

- Georgia's homeowner guide says the number of bedrooms determines septic tank
  size for a single-family residence.
- The guide says the minimum approved tank size is 1,000 gallons and that a
  3- or 4-bedroom house uses that baseline example.
- The guide says a house with a garbage disposal requires a septic tank 50
  percent larger.
- Peak daily water usage is described as 150 gallons per bedroom.

Operational interpretation:

- Georgia can support a stronger tank-size page than most states.
- The disposal modifier should be surfaced clearly, but still framed as a
  planning estimate until the underlying rule text is extracted directly.

## Pennsylvania

Why it is an anchor:

- DEP gives a clean homeowner explanation
- Local SEO and municipal enforcement role are explicit
- Good replacement and maintenance intent fit

Official points currently used:

- DEP says tank size is generally based on bedroom count.
- DEP gives 900 gallons or larger for a three-bedroom home as the public
  homeowner example.
- DEP says systems generally need pumping every three to five years, with large
  households needing it more often.
- DEP points homeowners to the municipality or local agency and the Sewage
  Enforcement Officer for permitting and enforcement.

Operational interpretation:

- Pennsylvania is good for a trust-first page because the local permit path is
  part of the value proposition, not a weakness.

## Connecticut

Why it is an anchor:

- Unique design-flow logic
- Potential-bedroom concept is unusual and valuable
- Local health authority role is explicit

Official points currently used:

- Connecticut says residential sewage disposal sizing is based on bedroom count,
  not actual occupant count.
- DPH uses 150 gallons per bedroom for residential design flow.
- New homes are based on potential bedrooms.
- Building additions and changes in use can trigger code-complying area review
  and soil testing.
- The local director of health or approved agent issues the approval to
  construct and the permit to discharge for most residential systems.

Operational interpretation:

- Connecticut should emphasize design-flow logic and permit sequence instead of
  trying to pretend the public tool can output exact gallon values yet.

## Oregon

Why it is an anchor:

- Site-evaluation-first flow is highly differentiating
- ADU and change-in-use paths create strong homeowner intent
- Great fit for estimate-first framing

Official points currently used:

- Site evaluation comes before permit certainty.
- DEQ says a site evaluation does not guarantee approval of a specific septic
  system type.
- Site evaluation reviews both the initial and replacement soil absorption
  areas.
- An authorization notice may be required for increased sewage flow, change in
  use, or an ADU connection to an existing system.
- A septic permit is required to install, alter, or repair a system and is
  valid for one year.

Operational interpretation:

- Oregon pages should lean hard into permit sequence, confidence bands, and
  site risk rather than pretend the main value is a single tank number.
